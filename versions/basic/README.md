# SystemSegure - Versión BASIC

## Diferencias con FREE

### Características BASIC
- ✅ Dashboard completo
- ✅ Reportes básicos
- ✅ **Gestión de usuarios** (hasta 5 usuarios)
- ✅ **3 dispositivos simultáneos**
- ❌ Analítica avanzada
- ❌ Acceso API
- ❌ Branding personalizado

### Límites
- **Usuarios máximos:** 5
- **Dispositivos máximos:** 3
- **Rate limiting:** 200 requests/15min

### Puertos
- **Backend:** http://localhost:3010
- **Frontend:** http://localhost:3011

### Instalación

Ejecutar script de instalación desde el directorio `scripts/`:

**Linux/macOS:**
```bash
chmod +x install.sh
./install.sh
```

**Windows:**
```cmd
install.bat
```

Para más detalles, ver el README completo de la versión FREE, las instrucciones son las mismas.
